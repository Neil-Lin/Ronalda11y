Ronalda11y Button

How to use

HTML

  1. Add the JavaScript file to HTML using the <script src="" defer> tag
  2. Use the <ry-btn> tag

Nuxt3

  1. Add the JavaScript file to the Nuxt3 project (possibly in the statics or assets folder).
  2. Add a `.client.js` file to the plugins folder with the following export: export default defineNuxtPlugin(() => { if (process.client) { import('your path') } })
  3. Add the plugin configuration to your nuxt.config.ts (or .js):plugins: ['~/plugins/rybtn.client.js']

Ensure to replace 'your-path' with the actual path to your JavaScript file. Also, make sure the path and filenames are correct to avoid loading errors.

Attribute

The Ronalda11y button inherits attributes, with some custom options available:
Name Description Available value
size Sets the size of the button. small, large
disabled Disables the button if set.
loading Disables the button and shows a loading animation when set.

Disabled

We do not recommend using the disabled and loading attributes because elements marked as disabled cannot be focused by the keyboard, and screen reading software will often ignore them. Additionally, the disabled state is typically designed to be less visually prominent to discourage attention or premature interaction. If possible, consider using a method that provides error feedback after an attempt to interact, rather than disabling the button. However, this concept has not yet become widely accepted, and there are indeed scenarios where using disabled is appropriate. Therefore, we have still designed it to be usable.
