Ronalda11y Breadcrumbs

How to use

HTML

  1. Add the JavaScript file to HTML using the <script src="" defer> tag
  2. Use the <ry-breadcrumbs> tag

Nuxt3

  1. Add the JavaScript file to the Nuxt3 project (possibly in the statics or assets folder).
  2. Add a `ry.client.js` file to the plugins folder with the following export: export default defineNuxtPlugin(() => { if (process.client) { import('your path') } })
  3. Add the plugin configuration to your nuxt.config.ts (or .js):plugins: ['~/plugins/ry.client.js']
  4. Add vite: { vue: { compilerOptions: { isCustomElement: (tag: string) => tag === 'ry-breadcrumbs' } } }
  5. Use the <ry-breadcrumbs> tag

Attribute

The Ronalda11y breadcrumbs inherits attributes, with some custom options available:
Name Description Type Default value
id Sets id string
class Sets classname string
aria-label Sets aria-label string
separator Sets the separator between breadcrumb items. string /
items Sets the text and url path of breadcrumb item. { "text":string, "url":string }
