Ronalda11y 麵包屑

如何使用

HTML

  1. 複製檔案 (breadcrumbs.js 或 ronalda11y.js 或 ronalda11y.min.js)
  2. 將 JavaScript 檔案引入至 HTML <script src="" defer>
  3. 使用 <ry-breadcrumbs> 標籤

Nuxt3

  1. 複製檔案 (breadcrumbs.js 或 ronalda11y.js 或 ronalda11y.min.js)
  2. 將 JavaScript 檔案放到 Nuxt3 專案。
  3. 在 plugins 資料夾新增 `ry.client.js` 檔案，內容範例如下： export default defineNuxtPlugin(() => { if (process.client) { import('your path') } })
  4. 在 nuxt.config.ts (或 .js) 增加 plugins:plugins: ['~/plugins/ry.client.js']
  5. 使用 <ry-breadcrumbs> 標籤

屬性

寫在 Ronalda11y 麵包屑上的屬性都會繼承，但以下屬性我們有做一點變化：
屬性名稱 說明 類型 預設值
id 設定 id string
class 設定 classname string
aria-label 設定 aria-label string
separator 設定分隔的符號 string /
items 設定麵包屑資料 { "text":string, "url":string }

設定一個你喜歡的 classname 並且在裡面改變麵包屑的 CSS Variables

CSS Variables

CSS Variables 名稱 說明 預設值
--ry-breadcrumbs-bg 麵包屑背景顏色 transparent
--ry-breadcrumbs-link-padding-vertical 麵包屑裡的連結左右間距 0.75rem
--ry-breadcrumbs-link-padding-horizontal 麵包屑裡的連結上下間距 0.5rem
--ry-breadcrumbs-text-size 文字大小 1rem
--ry-breadcrumbs-text-deco 文字底線 none
--ry-breadcrumbs-link-color 連結文字顏色 oklch(42.9% 0.2972777928415759 264.05202063805507)
--ry-breadcrumbs-link-visited-color 連結文字顏色(造訪過的連結) oklch(37.48% 0.167 303.51)
--ry-breadcrumbs-focus-shadow-width 聚焦時的邊框粗細 3px
--ry-breadcrumbs-focus-shadow-color 聚焦時的邊框顏色 oklch(83.15% 0.15681888825079074 78.05241467152487)
--ry-breadcrumbs-link-currentpage-color 最後一個，也就是目前頁面的文字顏色 oklch(13.98% 0 0)
