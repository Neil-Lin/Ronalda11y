Ronalda11y 勾選框

如何使用

HTML

  1. 複製檔案 (checkbox.js 或 ronalda11y.js 或 ronalda11y.min.js)
  2. 將 JavaScript 檔案引入至 HTML <script src="" defer>
  3. 使用 <ry-checkbox> 標籤

Nuxt3

  1. 複製檔案 (checkbox.js 或 ronalda11y.js 或 ronalda11y.min.js)
  2. 將 JavaScript 檔案放到 Nuxt3 專案。
  3. 在 plugins 資料夾新增 `ry.client.js` 檔案，內容範例如下： export default defineNuxtPlugin(() => { if (process.client) { import('your path') } })
  4. 在 nuxt.config.ts (或 .js) 增加 plugins:plugins: ['~/plugins/ry.client.js']
  5. 使用 <ry-checkbox> 標籤

如果發生錯誤，請先確保路徑以及名稱都正確。

屬性

Ronalda11y 勾選框有以下屬性可用：
名稱 說明
name 如果是在同一系列的選項，使用 name 表示
value 勾選框的值
checked 已勾選狀態
disabled 禁用狀態

問與答

相比原生如何？

使用 ry-checkbox 可以確保螢幕閱讀器正確閱讀 label 與 input 之間的關聯，推薦使用。

但即便是這樣還是不夠，如果是有一組多個相關 checkbox，最好搭配使用 role="group"、aria-labelledby、<ul>、<li>，如此一來可以知道同一組裡面有多少個 checkbox。
